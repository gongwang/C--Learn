========================================================================
    静态库：communication 项目概述
========================================================================

应用程序向导已为您创建了此 communication 库项目。

没有为此项目创建源文件。


communication.vcxproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，
    其中包含生成该文件的 Visual C++ 
    的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

communication.vcxproj.filters
    这是使用“应用程序向导”生成的 VC++ 项目筛选器文件。 
    它包含有关项目文件与筛选器之间的关联信息。 在 IDE 
    中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。
    例如，“.cpp”文件与“源文件”筛选器关联。

其他注释：

iconnectbase.h是使用静态库communication.lib的头文件

test目录内包含testserver 和 testclient。tcpserver 和 tcp连接方式可作为服务器和客户端发送消息， udp、com连接方式可互为客户端服务器。四中连接方式分别封装成一个.h文件。具体的调用顺序和调用方法可以参考.h。如不需要拆分详细的调用过程，可以直接使用.h，参考main函数的调用过程。
